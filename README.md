# как это сделать работать чтоб
1. скачаит
2. открыть в VSc оно само подсосет библиотеки
3. сконфигурировать если оно еще не сделало само
4. в терминале открыть папку build и выполнить по жести запустить cmake ..&& cmake --build .
5. полученную шляпу назвать client или какт похоже
6. поменять исполняемый файл с test1 на test2
7. снова собрать и скомпилировать
8. назвать его serv или какт похоже
9. запустить сначала серв потом клиент
10. происходит жеский обмен и все работает
