4. сконфигурировать если оно еще не сделало само
5. в терминале открыть папку build и выполнить по жести запустить cmake ..&& cmake --build .
6. в build будет три файла client, client_gui и serv
10. запустить сначала серв потом клиент (любой из 2)
11. происходит жеский обмен и все работает
