# как это сделать работать чтоб
1. скачаит
2. открыть в VSc оно само подсосет библиотеки
3. установить если нема: pkg-config и libwebkit2gtk-4.1-dev
4. сконфигурировать если оно еще не сделало само
5. в терминале открыть папку build и выполнить по жести запустить cmake ..&& cmake --build .
6. полученную шляпу назвать client или какт похоже
7. поменять исполняемый файл с test1 на test2
8. снова собрать и скомпилировать
9. назвать его serv или какт похоже
10. запустить сначала серв потом клиент
11. происходит жеский обмен и все работает
